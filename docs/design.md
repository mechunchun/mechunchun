## 1. 产品概述

### 1.1 产品名称
**Stock Indicator Lab** — 技术指标交互式计算与可视化工具

### 1.2 产品定位
一个基于浏览器的单页面工具，让用户能够：
- 选择不同股票的历史数据
- 实时计算并可视化 RSI、MACD、布林带、ATR 四项技术指标
- 动态调整每项指标的参数并立即重绘图表
- 无需编程基础，适合分析师、交易员、量化爱好者使用

### 1.3 核心价值
| 痛点 | 解决方案 |
|------|----------|
| 改参数要重新跑代码 | 滑块/输入框实时调整，图表立即更新 |
| 四个指标分开看，缺乏联动 | 4面板 Dashboard 同步展示 |
| 不知道参数改了有什么影响 | 每次调整参数，图表实时重绘，直观对比 |
| 数据格式不统一 | 内置标准化数据加载器，拖拽 CSV 即可 |

## 2. 用户故事

### 2.1 主要用户角色
- **散户投资者**：想用技术指标辅助买卖决策，不懂编程
- **量化分析师**：快速验证指标参数，做策略原型
- **金融学习者**：理解指标计算逻辑，通过交互加深理解

### 2.2 核心用户故事

**US-1**: 选择股票
> 作为一个用户，我想要从下拉菜单选择已上传的股票（或上传新 CSV），以便切换不同标的查看指标

**US-2**: 调整 RSI 参数
> 作为一个用户，我想要拖动滑块改变 RSI 的周期（6~30），以便找到最适合该股票的参数

**US-3**: 调整 MACD 参数
> 作为一个用户，我想要分别调整 MACD 的快线、慢线、信号线周期，以便对比不同参数组合的效果

**US-4**: 调整布林带参数
> 作为一个用户，我想要调整布林带的周期和标准差倍数，以便观察带宽变化

**US-5**: 调整 ATR 参数
> 作为一个用户，我想要调整 ATR 的周期和止损倍数，以便设置合适的止损位

**US-6**: 查看指标详情
> 作为一个用户，我想要点击某个指标面板查看详细数值表格，以便精确读取数据

**US-7**: 导出结果
> 作为一个用户，我想要导出计算后的数据为 CSV，以便在其他工具中继续使用

## 3. 技术架构

### 3.1 技术栈

```
前端框架:  纯 HTML + CSS + JavaScript（无框架依赖）
图表引擎:  Chart.js 4.x（Canvas 渲染，支持响应式）
计算引擎: 自定义 JavaScript 函数（不依赖后台）
数据格式:  CSV（UTF-8 BOM 兼容）
部署方式:  静态 HTML 文件，双击即可在浏览器打开
```

### 3.2 文件结构

```
stock_indicator_lab/
├── index.html              # 主页面
├── css/
│   └── style.css           # 样式文件
├── js/
│   ├── app.js              # 主逻辑
│   ├── indicators.js       # 四项指标计算函数
│   ├── chartManager.js     # Chart.js 图表管理
│   └── dataLoader.js       # CSV 数据加载器
├── data/                   # 预置股票数据
│   ├── 00981_HK.csv        # 中芯国际
│   ├── 01211_HK.csv        # 比亚迪
│   └── 600900_SH.csv       # 长江电力
└── README.md               # 使用说明
```

### 3.3 数据流

```
CSV 文件
   │
   ▼
dataLoader.js（解析、校验、标准化）
   │
   ▼
标准化 DataFrame（JavaScript Array of Objects）
   │
   ├──► indicators.js ──► 计算各指标
   │                      （RSI / MACD / BOLL / ATR）
   │
   ▼
指标数据（注入到原始数据数组）
   │
   ▼
chartManager.js（驱动 Chart.js 渲染）
   │
   ▼
Canvas 图表（4 个面板同步更新）
```

## 5. 功能详细设计

### 5.1 数据加载模块

**支持的数据格式**：

| 字段 | 必填 | 说明 |
|------|------|------|
| trade_date | ✅ | 交易日期，支持 YYYYMMDD 或 YYYY-MM-DD |
| open | ✅ | 开盘价 |
| high | ✅ | 最高价 |
| low | ✅ | 最低价 |
| close | ✅ | 收盘价 |
| vol | ❌ | 成交量（可选） |
| amount | ❌ | 成交额（可选） |

**数据来源**：
1. 预置数据：打包在 `data/` 目录的 3 只标的
2. 用户上传：拖拽或点击上传 CSV 文件
3. 在线获取（未来）：输入股票代码，调用 API 获取

### 5.2 指标计算模块

#### RSI 计算器

```javascript
/**
 * @param {number[]} closes - 收盘价数组
 * @param {number} period - 周期，默认 14
 * @returns {number[]} RSI 数组（前 period 个为 null）
 */
function calculateRSI(closes, period = 14) {
  // Wilder 平滑法
  // avg_gain = EMA(gain, period)
  // avg_loss = EMA(loss, period)
  // RS = avg_gain / avg_loss
  // RSI = 100 - 100 / (1 + RS)
}
```

**可调参数**：
- 周期：6 ~ 30，默认 14，步进 1
- 超买线：60 ~ 80，默认 70
- 超卖线：20 ~ 40，默认 30

#### MACD 计算器

```javascript
/**
 * @param {number[]} closes - 收盘价数组
 * @param {number} fast - 快线周期，默认 12
 * @param {number} slow - 慢线周期，默认 26
 * @param {number} signal - 信号线周期，默认 9
 * @returns {Object} { dif, dea, histogram }
 */
function calculateMACD(closes, fast = 12, slow = 26, signal = 9) {
  // DIF = EMA(fast) - EMA(slow)
  // DEA = EMA(DIF, signal)
  // histogram = (DIF - DEA) * 2
}
```

**可调参数**：
- 快线周期：5 ~ 20，默认 12
- 慢线周期：20 ~ 40，默认 26
- 信号线周期：5 ~ 15，默认 9

#### 布林带计算器

```javascript
/**
 * @param {number[]} closes - 收盘价数组
 * @param {number} period - 周期，默认 20
 * @param {number} mult - 标准差倍数，默认 2
 * @returns {Object} { middle, upper, lower, pctB, bandwidth }
 */
function calculateBollinger(closes, period = 20, mult = 2) {
  // middle = SMA(period)
  // std = population std(period)
  // upper = middle + mult * std
  // lower = middle - mult * std
  // %B = (close - lower) / (upper - lower)
  // bandwidth = (upper - lower) / middle * 100%
}
```

**可调参数**：
- 周期：10 ~ 50，默认 20
- 标准差倍数：1.5 ~ 3.0，默认 2.0，步进 0.1

#### ATR 计算器

```javascript
/**
 * @param {number[]} highs - 最高价数组
 * @param {number[]} lows - 最低价数组
 * @param {number[]} closes - 收盘价数组
 * @param {number} period - 周期，默认 14
 * @returns {Object} { tr, atr, atrPct, stopLong, stopShort }
 */
function calculateATR(highs, lows, closes, period = 14) {
  // TR = max(high-low, |high-prevClose|, |low-prevClose|)
  // ATR = EMA(TR, period)  // Wilder smoothing
  // stopLong = close - multiplier * atr
  // stopShort = close + multiplier * atr
}
```

**可调参数**：
- 周期：7 ~ 30，默认 14
- 止损倍数：1.0 ~ 3.0，默认 2.0，步进 0.5

### 5.3 图表交互设计

**重绘触发方式**：
1. 参数输入框：失去焦点或按 Enter 触发重绘
2. 滑块：停止拖动（onMouseUp）触发重绘
3. 重绘按钮：立即触发

**性能优化**：
- 使用 `requestAnimationFrame` 合并多次重绘
- 数据量大时（> 1000 条）使用数据降采样显示

### 5.4 数据表格

- 显示最新 20 条记录
- 支持点击表头排序
- 鼠标悬停高亮对应图表位置（未来）

## 6. 交互流程

### 6.1 主流程

```
用户打开页面
     │
     ▼
加载默认股票数据（中芯国际）
     │
     ▼
计算默认参数下的四项指标
     │
     ▼
渲染 4 个图表面板
     │
     ▼
用户调整某个指标的参数
     │
     ▼
实时重绘该指标面板（其他面板不变）
     │
     ▼
用户点击「重置」→ 恢复默认参数
```

### 6.2 参数调整流程

```
用户拖动 RSI 周期滑块（例如 14 → 20）
     │
     ▼
onMouseUp 事件触发
     │
     ▼
重新计算 RSI(20)
     │
     ▼
更新 RSI 面板数据
     │
     ▼
Chart.js update() 重绘（保留缩放状态）
```

## 9. 风险与应对措施

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| JavaScript 计算大量数据卡顿 | 用户体验差 | 使用 TypedArray，分片计算 |
| Chart.js 渲染 4 图性能问题 | 页面卡顿 | 降采样 + requestAnimationFrame |
| 用户上传数据格式不标准 | 解析失败 | 提供模板下载 + 格式校验提示 |
| 移动端图表太小看不清 | 可用性差 | 响应式 + 手势缩放 |
